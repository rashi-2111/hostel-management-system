##Hostel Management System

This project is a simple Hostel Management System built using Python.
It demonstrates file handling, data organization, and basic system logic without using a database.

---

#ModulesIncluded

- Student Management
- Room Allocation (Boys / Girls / Others)
- Visitor Records
- In-Time / Out-Time Tracking
- Leave Applications

---

#KeyFeatures

- Command-line based system
- File-based storage using ".txt" files
- Organized hostel data management
- Separate files for each module
- Beginner-friendly implementation

---

#TechnologiesUsed

- Python
- Text Files
